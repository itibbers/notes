关于阅读源码

拿 Vue 来说，上来就把源码 clone 下来阅读，会很吃力，因为很可能看不懂，不是看不懂语言，而是不懂为什么要这样写。所以阅读源码，需要先了解代码背后的设计思想，如果设计思想了解的充分了，源码甚至不用阅读都可以 。
